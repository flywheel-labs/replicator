# Replicator Artifact Schema

The neutral portability output is called a **Resonance Bundle**.

Each artifact has:

- `provider`
- `path`
- `artifact_type`
- `classification`
- `reason`
- `target_notes`
- `contains_secret_reference`

Classifications:

- `portable`
- `portable_with_edits`
- `manual_review`
- `not_portable`

The human-readable markdown output is called a **Resonance Report**.
